# LinuxProj
 这是一个用Makefile编译的简单的C项目。项目主要实现了排序功能：
 * 在终端输入1，对数组a中的元素进行插入排序；
 * 在终端输入2，对数组b中的元素进行冒泡排序。
 
 项目结构：<br />
  |—main.c  <br />
  |—sort.c  <br />
  |—Makefile  <br />
  |—include/  <br />
  |　|—sort.h  <br />
